
# FloraScan: Plant Identification and Information App

FloraScan is a mobile application designed to empower users with comprehensive plant information simply by scanning a picture of the plant using their smartphone's camera. With FloraScan, users can instantly access detailed data about various plants, including their benefits and uses.

## Features

- **Plant Recognition:** Utilizes advanced image recognition technology to identify plants accurately from images captured by the user's phone camera.
- **Comprehensive Plant Information:** Provides detailed information about each recognized plant, including its common and scientific name, native habitat, growth conditions, and maintenance requirements.
- **Beneficial Properties:** Users can explore the beneficial properties of each plant, such as medicinal uses, nutritional value, environmental benefits, and aesthetic appeal.
- **Uses and Applications:** Highlights the various uses and applications of each plant, ranging from culinary uses to herbal remedies, landscaping ideas, and ecological significance.
- **Personalized Recommendations:** Offers personalized recommendations based on user preferences, location, and plant-related interests.
- **Community Engagement:** Users can engage with the FloraScan community by sharing their own plant discoveries, tips, and experiences.

## Technology Stack

- **Frontend:** Flutter
- **Backend:** Node.js, Express, MongoDB
- **Model Training:** Python (using TensorFlow, scikit-learn)
- **Additional Libraries:** Node Mailer

## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your_username/FloraScan.git
   ```

2. Navigate to the project directory:

   ```bash
   cd FloraScan
   ```

3. Install dependencies for both frontend and backend:

   ```bash
   # Frontend (Flutter)
   cd frontend
   flutter pub get
   
   # Backend (Node.js)
   cd ../backend
   npm install
   ```

4. Run the backend server:

   ```bash
   # From the backend directory
   npm start
   ```

5. Run the Flutter app:

   ```bash
   # From the frontend directory
   flutter run
   ```
